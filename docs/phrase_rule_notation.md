# 熟語置換記法 追補

## 目的

`dictionary-rules` の `phrase_rules` を、従来の冗長な配列オブジェクト形式だけでなく、短縮記法でも扱えるようにする。

## 対応記法

### 従来形式

```js
phrase_rules: [
  {
    from: "奇跡",
    to: "奇蹟",
    priority: 90,
    enabled: true,
  },
]
```

### 短縮形式

```js
phrase_rules: {
  "奇跡": ["奇蹟", 90, true],
  "弁償": ["辦償", 90, true],
  "ドイツ": ["独逸,独乙", 90, true],
}
```

- 第1要素: 変換先
- 第2要素: `priority`
- 第3要素: `enabled`

## 複数候補

変換先にカンマ区切りで複数候補を与えた場合、エンジンはその候補群から 1 つを選ぶ。

```js
"ドイツ": ["独逸,独乙", 90, true]
```

この場合は、ページ URL と語句を seed にした擬似ランダム選択を行う。同一ページ内では同一語に同一候補を割り当てる。

## 設定画面との関係

設定画面の入出力も同じ記法を読める。保存時の出力も短縮形式を優先する。
